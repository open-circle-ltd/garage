========================
opencircle.garage Release Notes
========================

.. contents:: Topics

v1.0.0
======

Release Summary
---------------

Initial release. Provides modules to manage Garage S3 keys, buckets, and bucket-key permissions via the Garage Admin API v2.

New Modules
-----------

- ``opencircle.garage.garage_key`` - Manage S3 access keys.
- ``opencircle.garage.garage_bucket`` - Manage S3 buckets.
- ``opencircle.garage.garage_bucket_key`` - Grant or revoke S3 access key permissions on a bucket.
